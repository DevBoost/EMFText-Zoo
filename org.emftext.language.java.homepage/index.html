JaMoPP - The Java Model Parser and Printer

Overview

JaMoPP is a set of Eclipse plug-ins that can be used to parse Java source code into EMF-based models and vice versa.

Application areas

  • Using model transformations to process Java programs (e.g., using ATL)
  • Code generation
  • Code analysis
  • Code visualization (e.g., using GMF)
